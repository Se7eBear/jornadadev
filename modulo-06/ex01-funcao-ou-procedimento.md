Str() — converte número em string
QOut() — exibe texto na tela
Date() — retorna a data atual
Len() — retorna o tamanho de uma string ou array
Upper() — retorna a string em maiúsculas
ClearScreen() — limpa a tela

Str(): É uma função, pois processa o número fornecido e retorna o resultado como um novo valor em formato de texto (string).
QOut(): É um procedimento, pois sua única finalidade é executar a ação visual de imprimir o texto na tela, sem devolver um dado para ser usado no código.
Date(): É uma função, pois consulta o sistema operacional e retorna o valor exato da data atual para o programa.
Len(): É uma função, pois calcula e devolve um valor numérico (inteiro) que representa a quantidade exata de caracteres ou elementos.
Upper(): É uma função, pois recebe um texto e retorna uma nova string contendo o mesmo conteúdo convertido para letras maiúsculas.
ClearScreen(): É um procedimento, pois apenas realiza a ação de apagar o conteúdo do terminal, não gerando nenhuma informação ou valor de retorno.

Apesar que ate os procedimentos podem ser considerados uma funcão...
